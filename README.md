# 🍷 Red Wine Quality - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the Red Wine Quality dataset to understand the key factors that influence wine quality.

The goal is to extract meaningful insights from the data using visualization and statistical analysis techniques.

---

##  Objectives

* Understand the structure and distribution of the dataset
* Identify relationships between different features
* Discover key factors affecting wine quality
* Build strong data analysis intuition

---

##  Dataset Information

*  Source: UCI Machine Learning Repository
*  Total Records: 1599
*  Features: 11 input variables + 1 output variable (quality)

### 🧪 Features include:

* Fixed Acidity
* Volatile Acidity
* Citric Acid
* Residual Sugar
* Chlorides
* Free Sulfur Dioxide
* Total Sulfur Dioxide
* Density
* pH
* Sulphates
* Alcohol
* Quality (Target Variable)

---

## 🔍 Steps Performed

### 1️⃣ Data Loading & Understanding

* Loaded dataset using pandas
* Checked shape, data types, and summary statistics

### 2️⃣ Data Cleaning

* Checked for missing values
* Removed duplicate records
* Ensured dataset consistency

### 3️⃣ Exploratory Data Analysis

* Univariate analysis (distribution of features)
* Bivariate analysis (feature vs quality relationships)
* Correlation analysis using heatmaps

### 4️⃣ Data Visualization

* Histograms
* Boxplots
* Countplots
* Correlation Heatmap
* Pairplots (selected features)

---

##  Key Insights

*  **Alcohol** shows a strong positive relationship with wine quality
*  **Volatile Acidity** negatively impacts wine quality
*  **Sulphates** have a moderate positive influence
*  Most wines fall under **average quality (5–6 range)**
*  Some features show clear patterns affecting quality
