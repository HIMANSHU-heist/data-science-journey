# ✈️ Flight Price Analysis (EDA)

##  Overview

This project focuses on cleaning and analyzing flight data to understand patterns affecting ticket prices. It involves exploratory data analysis (EDA) and visualization to extract meaningful insights.

---

## 🔍 What This Project Covers

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Insight Extraction

---

## 📊 Key Insights

* Pricing varies significantly across different airlines
* Flight duration has a noticeable impact on ticket prices
* Certain routes are consistently more expensive
* Price distribution is skewed, with most tickets in a mid-range

---

## Tools Used

Python | Pandas | NumPy | Seaborn | Matplotlib

---

##  Project Structure

* `data/` → dataset
* `notebook/` → analysis notebook
* `images/` → visualizations
